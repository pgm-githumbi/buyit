# Cryptocurrency price tracking app with Expo react native

uses:
1. **expo file-based router**
2. **nativewind**
3. **reduxjs toolkit**
4. **rtk query**
5. **redux persist**

## Usage

- `git clone https://github.com/pgm-githumbi/buyit.git`
- `cd buyit`
- `npx expo install`
- _Run the code with:_
- `npx expo`
